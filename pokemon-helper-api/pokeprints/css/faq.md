

- Из чего сделана продукция?
- Все модели печатаются из PLA пластика (иногда из PETG) с высотой слоя 0.08мм, что позволяет добиться хорошей детализации для FDM печати

- Какие способы доставки доступны?
- Доставка осуществляется за счет покупателя любой службой доставки (Yandex, 5post, Ozon, CDEK и тд)

- Какие варианты упаковки доступны?
- Для самовывоза доступен вариант подарочной прозрачной упаковки. Для доставки только непрозрачная коробка с наполнителем для обеспечения безопасности перевозки

- Какого размера покеболы в виде покемонов?
- Основная часть покебола диаметром 7см, полный размер сильно отличается для каждого варианта покебола.

- Доступна ли какая-то скидка?
- При подписке на телеграмм канал скидка 5% на всю продукцию. Так же некоторые товары имеют скидку за опт.
